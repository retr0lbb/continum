# General Specifications for AI Agents

## 1. Reading All Specifications Before Any Execution
- Before executing any action or creating/modifying code, **agents must read all the files present in the `@src/specs/` folder**.
- This ensures that all actions align completely with the project’s established rules, guidelines, and expectations.

## 2. Asking Questions
- **If** any information is unclear or ambiguous, or if there are multiple possible ways of interpreting the task:
  - Agents must pause and ask the user questions to clarify the requirements.
  - Avoid making assumptions unless explicitly instructed by the user.

These principles are mandatory for ensuring both adherence to the project's standards and clear communication.
